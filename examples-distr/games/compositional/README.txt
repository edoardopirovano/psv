This example demonstrates the compositional functionalities of PRISM-games, including the support for multi-objective property specifications. See [BKW14] for the theoretical background of the assume-guarantee synthesis framework implemented in PRISM-games.

Executing the script comp_example.sh synthesises strategies both compositionally and monolithically.


For more information, see: http://www.prismmodelchecker.org/games/examples.php
